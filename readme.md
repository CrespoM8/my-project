# P3.1 Maquetación Responsive con SASS y Vite

> S2DAW Diseño de Interfaces Web
> Autor: Mario Crespo Alonso

## 1. Descripción

Maquetar las dos páginas (index y formulario) del diseño Figma, aplicando un diseño responsive y usando SASS con Vite. Conceptos de SASS utilizados: variables, mixins, y media queries.

---

## 2. Configuración y Tecnología

| Componente | Uso |
| :--- | :--- |
| **Generador** | Vite (Vanilla JS) |
| **Preprocesador CSS** | SASS (Dart Sass) |
| **Estructura HTML** | Multi-entry (index.html, formulario.html) |
| **Configuración SASS** | Uso de @use y @forward con _globals.scss para modularidad |
